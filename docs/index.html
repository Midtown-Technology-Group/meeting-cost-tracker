Meeting Cost Tracker

Know what your meetings really cost

Calculate the true cost of Microsoft Teams meetings based on participant time and hourly rates. A 60-minute meeting with 8 people at $150/hr costs $1,800. Get the data to justify fewer, shorter meetings.

View on GitHub Get Started

Features

💰

Cost Calculation

Automatically calculates meeting cost: duration × attendees × hourly rate. See the real price tag of every meeting.

📊

Microsoft Graph Integration

Pulls calendar data from Outlook/Teams. No manual data entry. Works with your existing calendar.

⚙️

Flexible Cost Models

Per-person rates, role-based pricing, or organization-wide defaults. Configure once, analyze forever.

📈

Rich Analytics

Top expensive meetings, attendees by cost, daily trends, organization breakdown. Spot the time sinks.

💡

Savings Analysis

"What if meetings were 30 minutes?" or "15 minutes shorter?" Quantify the savings from optimization.

📤

Export Reports

CSV or Excel export with charts. Share with clients, management, or use for internal billing.

Installation

From Source

git clone https://github.com/Midtown-Technology-Group/meeting-cost-tracker.git cd meeting-cost-tracker pip install -e .

First Run

# Authenticate with Microsoft Graph mct analyze --weeks 1 # Configure rates code ~/.meeting-cost-tracker/config.yaml

Configuration

~/.meeting-cost-tracker/config.yaml
costs: default_rate: 100.0 # Default hourly rate # Per-person rates person_rates: john.smith@company.com: 150.0 jane.doe@company.com: 200.0 # Role-based rates role_rates: engineer: 100.0 manager: 150.0 director: 200.0 c_level: 300.0 # Organization rates (by domain) org_rates: company.com: 125.0 client.com: 150.0

Usage Examples

Analyze Last 4 Weeks

mct analyze --weeks 4

Custom Date Range

mct analyze --start 2026-01-01 --end 2026-01-31

Override Default Rate

mct analyze --weeks 2 --rate 150

Export to Excel

mct analyze --weeks 4 --export report.xlsx

Show Savings Analysis

mct analyze --weeks 4 --savings

View Configuration

mct config

Cost Formula

Meeting Cost = Duration (hours) × Σ Attendee Hourly Rates
Example: 60-minute meeting with 8 people at $150/hr = $1,800
