# Meeting Cost Tracker - Project Site

A simple, single-page project site for Meeting Cost Tracker. Designed for GitHub Pages.

## Deploy to GitHub Pages

1. In your GitHub repo, go to **Settings → Pages**
2. Under "Build and deployment" select **Deploy from a branch**
3. Select `master` branch and `/docs` folder
4. Click Save

The site will be live at:
https://midtown-technology-group.github.io/meeting-cost-tracker/

## Local Preview

Simply open `index.html` in a browser, or use a local server:

```bash
cd docs
python -m http.server 8000
```

## Customization

- Edit colors in `styles.css` (CSS variables at the top)
- Replace logo SVG in `index.html` with your own
- Update feature descriptions, examples, etc.
